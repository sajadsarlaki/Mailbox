# Mailbox


## Usage
 1- clone project
 <br>
 2- cd DB_React
 ### back
 1- cd back 
 <br>
 2- run ```
       yarn
       ``` 
in terminal.
 <br>
 3- run ```
       yarn start
       ``` 

### front
 1- cd front
 <br>
 2- run ```
       yarn
       ``` 
in terminal.
 <br>
 3- run ```
       yarn start
       ```
